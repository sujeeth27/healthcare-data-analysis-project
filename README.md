# NHANES Body Measurements Analysis

This project analyzes anthropometric data from the NHANES Body Measures dataset using Python.

The project demonstrates:

- Data cleaning
- Exploratory data analysis
- Data visualization
- Feature engineering
- Linear regression using Scikit-Learn
- Model evaluation

#Uses:

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn


#Skills:

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Machine learning
- Data visualization
